# Code Of Conduct

This project adopts the **Contributor Covenant v2.1** as its code of conduct.

The canonical text is at: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

It is also available in many translations at: https://www.contributor-covenant.org/translations/

## Our Pledge In Brief

We commit to making participation in this project a respectful experience for everyone, regardless of background, identity, or experience level. We welcome contributors who engage constructively, give and receive feedback gracefully, and focus on what is best for the project and the community.

The full Contributor Covenant text covers expected behaviour, unacceptable behaviour, enforcement responsibilities, scope, and the enforcement process. Please read it in full before contributing.

## Scope

This code applies in all project spaces — the GitHub repository, issue tracker, pull request reviews, project chat channels, and any public spaces where someone is representing the project.

## Reporting

If you experience or witness conduct that violates the Contributor Covenant, please report it to the project maintainers.

**Reporting contact:** *to be added when the project hosts public infrastructure.*

Reports are handled confidentially. The reporter's identity will not be shared with the reported party without consent unless required by law.

## Enforcement

Maintainers are responsible for clarifying and enforcing this code. They may take any action they deem appropriate in response to violations, ranging from a private clarification to a permanent ban from the project, following the Contributor Covenant's "Enforcement Guidelines" section.

Enforcement decisions are not subject to public debate. A contributor who believes an enforcement action was made in error may appeal once, in writing, to the maintainer group.
